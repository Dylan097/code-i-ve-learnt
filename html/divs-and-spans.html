Divisions

A division, or div, is something that seperates a section so 1 section can have multiple different looks

A div is like a paragraph, except you can't add text to a div, but each new div will be below the last

Spans

A span is different to a div. A span can be nested inside a paragraph, so that the paragraph can have different looks to different words.

When you create multiple spans, until there's a new paragraph created, each span will be side-by-side in a web browser
